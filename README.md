# Blockchain Basics  
A beginner-friendly repository to learn blockchain technology, including fundamental concepts, smart contracts, and decentralized applications (DApps).  

## What is Blockchain?  
Blockchain is a decentralized and distributed ledger technology that ensures secure, transparent, and tamper-proof transactions.  

## Features  
- Learn blockchain fundamentals  
- Understand how decentralized networks work  
- Explore smart contracts using Solidity  
- Build simple blockchain applications using Python
